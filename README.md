# gizmogui

## Install Project & Dependencies
```
cd ~/gizmogui/
npm install
```

## For Development
### Compiles , Hot-Reloads , Serves
```
cd ~/gizmogui/
npm run serve
```

## For Production
### Compiles , Minifies
```
cd ~/gizmogui/
npm run build
```

## Lints & Fixes Files
```
cd ~/gizmogui/
npm run lint
```

## Serve Project
### 'serve' May Already Be Installed
```
cd ~/gizmogui/
npm install serve
npx serve -n -l tcp://0.0.0.0:8080 ./
```
